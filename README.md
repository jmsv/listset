## Getting Started

### Install

```bash
pip install listset
```

### Import and Use

```python
>>> from listset import listset

>>> listset([1, 2, 3, 4, 4, 4])
[1, 2, 3, 4]

>>> listset(['apple', 'orange', 'banana', 'apple', 'apple'])
['apple', 'orange', 'banana']
```
